# 🏝️ Travel Website

A modern, responsive travel-themed landing page created using **HTML5**, **CSS3**, and **JavaScript**. This project showcases a beautiful layout with smooth scrolling, elegant visuals, and a fully responsive design — perfect for travel agencies or portfolio demonstrations.

🚀 **Live Demo:**  
👉 [Click here to visit the live website](https://sundas-razzaq.github.io/Travel-Website/)

## 🌍 Features

- 🧭 Responsive navigation bar
- 🎞️ Full-screen hero section with call-to-action
- 🖼️ Image gallery to showcase destinations
- 📱 Fully responsive for mobile, tablet, and desktop
- 💡 Simple animations and hover effects
- 🔗 Social media footer

## 🛠️ Tech Stack

- **HTML5** – Semantic structure
- **CSS3** – Layout, design, media queries
- **GitHub Pages** – Live deployment
